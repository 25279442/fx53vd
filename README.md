# 9.1 更新：

#### EFI - 9.1 功能性更新包含对驱动的修改更新。

##### 修复的问题
```
1. 解决了SD卡读取硬件的驱动问题.

```
##### Clover EFI
```

```
#### 安装 EFI - 9.1 更新会将：
```
1. 上方提到的各种驱动更新到最新.
```
#### 要更新你 Hacintosh 上的“EFI V9.1”版本补丁，请进行以下操作：
1. 使用Clover挂载你的固态硬盘或者机械硬盘上的EFI分区.
2. 将你的SMBIOS信息或整个EFI目录保存至其他地方备份.
3. 将下载的EFI - 9.1.zip文件解压至根目录,保证文件夹名为"EFI".
4. 将你的SMBIOS数据还原到EFI V9.1的config文件中,主要是序列号和三码.如果你不知道什么是SMBIOS数据还原/备份,请无视这句话或手动随机生成一个机器数据.
5. 确认所有文件准备就绪:EFI分区中存在"EFI"文件夹,"EFI"文件夹中应该存在"CLOVER"与"BOOT"文件夹.其中"CLOVER"文件夹中存储本次更新后的文件归档.
5. 重启使用.
