# 8.9.4 更新：
发布于 2019 年 2 月 20 日

#### EFI - 8.9.4 功能性更新包含对驱动的更新。

##### DSDT
```
1. 将提取出的DSDT作为默认ACPI表加载.
```
##### 驱动程序
```
1. 更新了AppleALC到1.3.5版本.
2. VirtualSMC更新到1.0.2.
3. 加入了HoRNDIS.kext驱动以支持安卓手机USB网络共享.
4. WhateverGreen.kext的更新.
```
#### 安装 EFI - 8.9.4 更新会将：
```
1. 上方提到的各种驱动和DSDT更新到最新.
```
#### 要更新你 Hacintosh 上的“EFI V8.9.4”版本补丁，请进行以下操作：
1. 使用Clover挂载你的EFI分区.
2. 将你的SMBIOS信息或整个EFI目录保存至其他地方备份.
3. 将下载的8.9.4.zip文件解压至根目录,保证文件夹名为"EFI".
4. 重启.

# 8.9.3#Patch1 补充更新：
发布于 2018 年 10 月 16 日

#### EFI - 8.9.3#Patch1 补充更新包含对内置声卡的功能性改进，解决了耳机音频相关的问题。

##### 音频
```
适用于：     EFI - 8.9.3,运行macOS Mojave的系统，历代EFI版本在华硕FX53VD机器上存在的 Hackintosh 声卡问题。
影响：       开机时耳机孔需要睡眠一次才可以消除背景音
描述：       已通过重新整理声卡Codec并基于vit9696的官方库重新编译声卡驱动解决了开机时耳机背景音的问题。
补丁贡献者：  来自扬州大学 机械工程学院 的 QiuChenly 
```
#### 安装 EFI - 8.9.3#Patch1 补充更新会将：
```
1. “AppleALC.kext” 从 1.2.8 版本更新至版本 1.3.3 (基于Vit9696源码重新完整编译)。
2. “Config.plist”内关于声卡属性注入layoutID从“28”更新到“15”(ID将会提交给AppleALC开源项目，此ID用于ASUS FX53VD i5 7500 HQ/i7 7700 HQ机器)。
```

# 8.9.3 更新(未包含8.9.3#Patch1 补充更新)：
- SSDT/DSDT:
1. 增加了SSDT-ELAN.aml，SSDT-RALS.aml，SSDT-ATK-KABY.aml，针对触控板和背光键盘做一定的基础工作，后期将修复键盘背光和fn键完整功能。
- 其他驱动:
1. lilu内核调试输出开启。
2. 屏幕背光终于不用放到l/e下面了，可直接驱动16档亮度调节。支持10.12+
3. 使用新fakesmc技术，修正了acpi电池不变电量的问题。
4. 使用新CPUFriend.kext睿频技术，使cpu在800mhz-3.41Ghz下工作，有效节省电量。
5. 触控板更新驱动，使用更灵活了
6. 睡眠后可以在交流电的情况下使用usb键盘唤醒机器了

# 8.9.2 更新：
- 引导:

- DSDT:
